# Influencer Analytics for Brand Promotions
Live : https://sxresh.github.io/Influencer-Analytics-/
## Project Overview

This project analyzes influencer marketing data to help brands identify high-performing influencers, evaluate engagement effectiveness, and estimate promotional revenue opportunities across different niches.

The project includes:
- Data Cleaning and Analysis using Python
- Interactive Power BI Dashboard
- Streamlit Web Dashboard
- KPI Monitoring and Business Insights
- Niche-wise Performance Analysis

## Objectives

- Analyze influencer engagement and follower metrics
- Compare influencer performance across niches
- Identify high-value influencers for brand promotions
- Estimate expected promotional revenue
- Generate actionable business insights through interactive dashboards

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Streamlit
- Plotly
- GitHub

## Dashboard Features

- Total Followers KPI
- Average Engagement Rate
- Average Influencer Score
- Expected Revenue Analysis
- Niche Filter
- Followers vs Engagement Rate Analysis
- Influencer Score by Niche
- Interactive Visualizations
- Business Insights Section

## Key Insights

- Fitness influencers achieve the highest average influencer score.
- Health influencers demonstrate strong promotional potential.
- Technology influencers show the lowest average influencer score.
- Engagement rates generally decrease as follower counts increase.
- Fitness and Health niches generate the highest expected revenue opportunities.

## Project Outcome

The dashboard enables brands and marketing teams to make data-driven decisions when selecting influencers for promotional campaigns and maximizing campaign effectiveness.
